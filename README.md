# Welcome to BrainLog

BrainLog is a web app to help track your daily **mental health** and **wellness level**.  

## Features  (Work in Progress)

 - Slider to select your wellness today on a scale (with emojis)
 - Select up to 3 main emotions (From emotions wheel)
 - A space to add comments about what is  the main worry at that moment
 - A text box to add things you could work on / do better tomorrow
